== Description ==

The ATX GD Theme was designed and constructed solely for the ATX Game Designers group website. The styling of the theme is focused on providing presentable static web pages and customized appearances for WPForms and the Pretty Google Calendar plugins. Improvements, updates, and any initial redistribution of the theme after the completion of this project is at the discretion of ATX Game Designers: atxgamedesigners@gmail.com
